# Mods

Place each Teardown mod in its own subdirectory, for example:

```
mods/
  README.md          ← this file
  space-station/     ← example future mod
    info.txt         ← or equivalent manifest for your Teardown version
    main.lua
    ... assets ...
```

Add a short `README.md` inside each mod folder with:

- Teardown version tested
- Install location on Windows (and macOS/Linux if applicable)
- Controls or custom keybinds
- Known limitations
